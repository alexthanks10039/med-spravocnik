# MED SPRAVOCHNIK Flutter

Адаптивный Flutter-клиент медицинского справочника для web, Android и iOS. Интерфейс ориентирован на быстрый клинический поиск, структурированное чтение и работу с сохранёнными материалами.

## Реализовано

- адаптивная навигация: `NavigationBar` на смартфоне и `NavigationRail` на широком экране;
- главная, поиск, каталог и детальные страницы;
- заболевания, препараты, рекомендации и калькуляторы;
- BMI и eGFR CKD-EPI 2021;
- избранное, история, заметки, профиль и настройки;
- светлая, тёмная и системная темы;
- offline mock-репозиторий медицинских материалов.

## Стек

- Flutter / Dart;
- Material 3;
- Riverpod для зависимостей и состояния;
- GoRouter для маршрутизации;
- SharedPreferences для локальных пользовательских настроек.

## Запуск

```bash
flutter pub get
flutter run -d chrome --web-port 8080
```

Для Android или iOS выберите доступное устройство через `flutter devices`, затем выполните `flutter run -d <device-id>`.

## Проверки

```bash
flutter analyze
flutter test
```

## Архитектура

```text
lib/
├── app/                 приложение и маршрутизация
├── core/
│   ├── data/            репозиторий и Riverpod providers
│   ├── models/          предметные модели
│   └── theme/           темы и их сохранение
├── features/            экраны по функциональным областям
└── shared/widgets/      переиспользуемые UI-компоненты
```

Основная абстракция данных — `MedicalRepository`. `OfflineMedicalRepository` содержит демонстрационные записи и имитирует задержку поиска. Production-реализация должна подключать REST API, локальную БД и фоновую синхронизацию, не меняя presentation-слой.

## Маршруты

`/`, `/search`, `/catalog`, `/diseases`, `/drugs`, `/calculators`, `/articles`, `/saved`, `/history`, `/notes`, `/profile`, `/settings`, `/detail/:id`.

## Ограничения MVP

- клиент пока использует локальные данные и не вызывает backend;
- сохранённые материалы и заметки рассчитаны на локальный сценарий;
- медицинские тексты являются демонстрационными;
- перед релизом нужны источники, дата пересмотра, evidence grading и профессиональная редактура.
