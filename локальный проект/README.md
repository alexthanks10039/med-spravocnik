# Локальный проект

> TODO: позже разобраться с переносом и структурой локальной версии проекта.

Эта папка зарезервирована для отдельной локальной версии медицинского справочника.

Перед переносом необходимо:

- исключить `.venv`, кэши, Flutter `build` и другие генерируемые файлы;
- удалить абсолютные пути `C:\\...` и `D:\\...` из конфигурации;
- не публиковать SQLite-корпус, исходные медицинские документы и локальные журналы;
- определить, включать ли сервис калькуляторов как код, submodule или отдельную зависимость;
- подготовить безопасные `.env.example`, `.gitignore` и инструкции запуска;
- проверить лицензии и допустимость публикации медицинских материалов.

Текущий рабочий проект на компьютере этой заглушкой не изменяется.
